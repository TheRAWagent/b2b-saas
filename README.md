# B2B SaaS Application

This is a B2B SaaS application built with a modern stack, featuring a React frontend and a FastAPI backend.

## Project Structure

- **[frontend](./frontend)**: The user interface built with React, TypeScript, Vite, and TanStack Router.
- **[backend](./backend)**: The API server built with Python, FastAPI, and SQLAlchemy.
